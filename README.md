Olá.
Automação em Python real utilizada no setor de patrimônio da Sociedade Franciosi (Empresa de agronegócio ).
Essa automação consiste em minimizar o trabalho repetitivo e erros humanos de cadastros de equipamentos no site "PIMS", DA EMPRESA "TOTVS".
Utilizei selenium para conseguir navegar na web e poder fazer a entrada de dados através de codigos html e css.
No final, mais de 5 mil ativos foram cadastrados em minutos, fazendo com que o setor pudesse focar outros trabalhos invés de cadastrar um por um manualmente.
